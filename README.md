## 🚀 How to Use

### Requirements
```bash
pip install pandas numpy matplotlib seaborn
```

### Run Analysis
```bash
python sales_analysis.py
```

### Outputs Generated
- Console output with key metrics
- `comprehensive_analysis_report.txt` - Full statistical report
- Visualization PNG files

### Customize Analysis
```python
from sales_analysis import SalesAnalyzer

# Initialize with your data
analyzer = SalesAnalyzer('your_data.csv')

# Run specific analyses
regional = analyzer.analyze_regional_performance()
products = analyzer.analyze_product_performance()
trends = analyzer.analyze_time_trends()

# Export comprehensive report
analyzer.export_analysis_report('custom_report.txt')
```
